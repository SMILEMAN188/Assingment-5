Question-2: What is the difference between innerHTML, innerText, and textContent?

(innerHTML) set or get text HTML including tags,(innerText) gets visible text, and (textContent) gets all text including hidden

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a JavaScript technique where a single event listener is attached to a parent element to handle events for multiple child elements

Question-4: What is event bubbling in the DOM?

In the DOM, event bubbling is a process where an event triggered on a specific element "bubbles up" or propagates to its parent, grandparent, and ancestor elements in the DOM tree until it reaches the root
